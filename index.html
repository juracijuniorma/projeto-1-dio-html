Entendendo a Comunicação Client x Server

Voltar

Lado Client - Lado cliente, também conhecido como front-end, refere-se às operações que são realizadas pelo cliente em um relacionamento cliente-servidor em uma rede de computadores.

Lado Server - Lado do servidor, também conhecido por back-end, é um termo usado para designar operações que, em um contexto cliente-servidor, são feitas no servidor, não no cliente.

Hospedagem de Sites

Voltar

Hospedagem de Site na Web - A hospedagem na Web oferece ao seu site ou aplicação da Web um servidor ou uma máquina na Internet. Porém, um nome de domínio fornece aos seus aplicativos um endereço para que seus usuários possam encontrar você.

Qual a diferença entre Hospedagem e Domínio de Site na Web - O domínio é apenas o endereço do seu site, enquanto a hospedagem é o que armazena todos os dados do seu site. Sem a hospedagem de sites, seu site simplesmente não existe e seu domínio iria apontar para nenhum lugar.

Linguagem de Programação

Voltar

Linguagem de Programação - A linguagem de programação é um método padronizado, formado por um conjunto de regras sintáticas e semânticas, de implementação de um código fonte que pode ser transformado em um programa de computador, ou usado como script interpretado que informará instruções de processamento ao computador.

Linguagem de Programação (Client-Side) - A linguagem do lado cliente ou Client-Side scripting. É uma linguagem que é executada no lado cliente, ou seja, no computador do próprio usuário. A tradução ou processamento é realizado pelo navegador da web, onde já existe uma biblioteca capaz de traduzir todos os comandos da página usando scripts do lado do cliente.

Linguagem de Programação (Server-Side) - Já a linguagem do lado servidor ou Server-Side scripting. É uma linguagem cujo processamento é feito no servidor, o que significa que o servidor da Web foi integrado por um mecanismo da Web. Aqui, o papel do mecanismo da web é processar todos os scripts existentes inclusos na categoria de script do lado do cliente.

Primeira parte do conteudo Sobre HTML

Voltar

O que é HTML e para que serve - HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. Essa linguagem de marcação HTML, junto com o CSS e o JavaScript, é fundamental para a estruturação de um website, do mais simples ao mais complexo.

Onde é usado o HTML - É usada em: Desenvolvimento frontend, ou seja, na parte visual das aplicações web; Desenvolvimento de aplicativos mobile, tanto para Android como para iOS; Desenvolvimento de jogos.

Obs:

É importante entender que ele não é uma linguagem de programação, já que seu objetivo não é criar um software, e, sim, marcar a estrutura de linguagens de programação de fato, como exemplo Javascript. Ou seja, ele indica onde os elementos aparecerão na aplicação.

Estrutura basica do HTML - O documento HTML sempre inicia com o que chamamos de estrutura básica. Esta estrutura é quase que imutável. Sempre será dessa forma e você sempre, sempre começará seu HTML começando por esse código.
